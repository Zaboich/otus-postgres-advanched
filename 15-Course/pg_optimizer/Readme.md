# Скрипт поиска оптимальной конфигурации Postgresql

* Установка venv в проекте
```
python -m venv .venv
```
* Активация venv
```
source .venv/bin/activate
```
* Установка зависимостей
```
pip install -r requirements.txt
```

## Настройка и запуск
1. Проверить и настроить конфигурацию `config.yaml`
2. Запустить скрипт
./pg_tuner.py
